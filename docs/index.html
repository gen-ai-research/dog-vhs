Canine Cardiac Keypoint Detection (CCKD)

The Canine Cardiac Keypoint Detection (CCKD) project introduces a novel, meticulously curated dataset for vertebral heart score (VHS) prediction in dogs. The CCKD model, featuring a hybrid Mamba-Residual architecture, achieves state-of-the-art performance in keypoint detection accuracy and VHS estimation. This model is designed to support efficient training and accurate predictions, setting a new benchmark for canine cardiology research.

For more information, pretrained models, and the annotation tool, please visit the official project repository at GitHub.
