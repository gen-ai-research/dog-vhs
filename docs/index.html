Canine Cardiac Keypoint (CCK) Dataset

The CCK dataset is a meticulously curated collection of 21,465 annotated canine thoracic radiographs, capturing a diverse range of anatomical variations and clinical conditions. This high-quality dataset provides a robust foundation for training deep learning models for vertebral heart score (VHS) estimation. It is designed to improve the accuracy and generalizability of machine learning models by offering comprehensive anatomical coverage. The dataset is available for download with detailed documentation on data structure, annotation protocols, and usage guidelines.

Installation and Setup

Download the Dataset from the given link and extract it into static folder for VHSMarker Annotation Tool. Download the Dataset from the given link and extract it into data folder for MambaVHS Model.

VHSMarker Tool

VHSMarker is a web-based human-computer interaction annotation tool that streamlines the creation of high-quality ground truth data for canine cardiac analysis. It provides an intuitive interface for precise labeling of six critical cardiac keypoints in dog heart X-ray images, significantly reducing annotation time and improving consistency. The tool supports real-time VHS calculation, side-by-side comparison of ground truth and model predictions, and automated data export for efficient model training.

Installation and Setup

cd vhs_marker_tool
npm install

Running

python app.py

MambaVHS Model

MambaVHS (Mamba-Enhanced Vertebral Heart Score Detection) is a custom deep learning model that integrates Mamba blocks for efficient long-range sequence modeling alongside complementary convolutional components for precise spatial feature extraction. It is specifically designed to capture both local anatomical features and global contextual information, making it particularly effective for complex veterinary diagnostics. The model's architecture is optimized for accurate VHS prediction with minimal latency, providing a scalable solution for real-world veterinary applications.

Installation and Setup

cd mamba_vhs/models
pip install -r requirements.txt

Training

python model.py
