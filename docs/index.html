VetMark and MambaVHS - Canine Cardiac Keypoint Detection

Dog Cardiomegaly is a serious medical condition characterized by an enlarged heart, which can significantly impact a dog's health and quality of life. Early and accurate detection of cardiomegaly is crucial for effective treatment and management. The vertebral heart score (VHS) is a widely used metric for assessing heart size in dogs from thoracic X-rays.

VetMark - Advanced Annotation for Cardiac Analysis

We present VetMark, a web-based annotation tool designed to streamline the creation of high-quality ground truth data for canine cardiac analysis. VetMark provides an intuitive interface for precise labeling of six critical cardiac keypoints in dog heart X-ray images, significantly reducing annotation time and improving consistency. It also supports:

  • Real-time vertebral heart score (VHS) calculation as points are placed.
  • Predictions using pretrained models for rapid evaluation.
  • Side-by-side comparison of ground truth and model predictions for performance analysis.

Canine Cardiac Keypoint (CCK) Dataset

The CCK Dataset is a meticulously curated collection of annotated dog heart X-rays specifically designed for accurate VHS prediction and keypoint localization. This dataset is an essential resource for developing robust diagnostic tools for veterinary cardiology, including early detection of Dog Cardiomegaly.

MambaVHS - Advanced Keypoint Detection Model

MambaVHS is a custom deep learning model that integrates Mamba blocks for efficient long-range sequence modeling alongside complementary convolutional components for precise spatial feature extraction. It is designed to surpass existing baselines in canine cardiac keypoint detection, enabling accurate VHS estimation for early cardiomegaly diagnosis.

Features and Benefits

  • Accurate keypoint detection and VHS estimation for Dog Cardiomegaly assessment.
  • Real-time feedback during annotation.
  • Model predictions compared with ground truth for validation.
  • Open-source tools and pretrained models for community use.
